# University-MongoDB

Proyecto de base de datos no relacional MongoDB para la gestión de información universitaria.

## Estructura actual
- **Colección de estudiantes**: Incluye datos personales, académicos, extracurriculares, cursos y financiamiento.
- **Colección de docentes**: Contiene datos personales, académicos, cursos dictados, horarios y publicaciones.
- **Colección de cursos**: Información de asignaturas, códigos, departamentos y créditos.

## Funcionalidades implementadas
- Inserción de múltiples estudiantes, docentes y cursos.
- Consultas avanzadas sobre estudiantes por carrera, cursos, estado académico, fraternidades, etc.
- Ejemplos de operaciones CRUD: inserciones, búsquedas, actualizaciones y eliminaciones.
- Uso de agregaciones y filtros para reportes académicos y extracurriculares.

## Ejemplo de uso
El proyecto incluye scripts con ejemplos de inserciones y consultas típicas para cada colección, facilitando la gestión y análisis de datos universitarios en MongoDB.
